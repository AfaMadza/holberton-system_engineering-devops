# SSH

## Description
By the end of this project, students should be able to explain and understand:

* What is a server
* Where server usually live
* What is SSH
* How to create an SSH RSA key pair
* How to connect to a remote host using an SSH RSA key pair

## Concepts
* ``ssh``, ``ssh-keygen``, ``env``
